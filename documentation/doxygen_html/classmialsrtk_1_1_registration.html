Detailed Description

template<typename TImage>
class mialsrtk::Registration< TImage >

This is a base class for doing registration.

Author
Marc Schweitzer, modified by Sebastien Tourbier - Enable more control on the choice of metrics, optimization parameters, including min/max Step Length, gradient magnitude tolerance, relaxation factor. ()
