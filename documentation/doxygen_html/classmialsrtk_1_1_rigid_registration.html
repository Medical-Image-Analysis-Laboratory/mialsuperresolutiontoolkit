Detailed Description

template<typename TImage>
class mialsrtk::RigidRegistration< TImage >

Preconfigured rigid registration.

This class encapsulates registration code common to many applications/classes of the library (metric, optimizer, transform, interpolator). The used components provided good results in fetal brain imaging, and therefore were kept fixed.

Author
Estanislao Oubel, Marc Schweitzer, Sebastien Tourbier

typedef itk::CenteredTransformInitializer< RigidTransformType, ImageMaskType, ImageMaskType > mialsrtk::RigidRegistration< TImage >::TransformInitializerType

Type of the transform initializer.

The transform initializer is used to provide an initial rigid transformation computed from the image masks. This is important when a large misalignment is introduced by fetal movement and images fall outside of the metric's capture region.
